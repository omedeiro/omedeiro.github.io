# Activity Analysis

## Overview
A comprehensive activity analysis tool that processes and visualizes fitness data to provide insights into running patterns, performance trends, and geographical activity mapping.

## Technologies Used
- Python
- Data Analysis (Pandas, NumPy)
- Visualization (Matplotlib, Plotly)
- Geospatial Analysis
- GPS Data Processing

## Features
- **GPS Route Mapping**: Visualize running routes with detailed geographical data
- **Performance Analytics**: Track pace, distance, and elevation changes
- **Activity Trends**: Analyze patterns over time
- **Interactive Visualizations**: Dynamic charts and maps

## Installation & Usage
```bash
git clone https://github.com/omedeiro/activityAnalysis.git
cd activityAnalysis
pip install -r requirements.txt
python analyze_activity.py
```
