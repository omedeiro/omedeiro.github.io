About Me

I’m a passionate software developer with experience spanning data analysis, scientific computing, virtual reality, and financial technology. I enjoy tackling complex problems across diverse domains, from physics simulations to interactive VR experiences and data-driven applications.

Background¶

My journey in software development has taken me through various fascinating domains:

  • Scientific Computing: Implementing complex physics simulations for superconductor research

  • Data Analysis: Processing GPS tracking data and financial datasets for insights

  • Virtual Reality: Creating immersive experiences with haptic feedback systems

  • Financial Technology: Building tools for personal finance management and analysis

Technical Expertise¶

Programming Languages¶

  • Python: Primary language for data analysis, scientific computing, and automation

  • C#: Unity development and Windows applications

  • JavaScript: Web development and interactive visualizations

  • MATLAB: Scientific computing and algorithm development

Frameworks & Libraries¶

  • Data Science: Pandas, NumPy, SciPy, Matplotlib, Plotly

  • Web Development: Streamlit, Flask, HTML/CSS

  • Game Development: Unity 3D, SteamVR

  • Scientific Computing: Custom numerical solvers and simulation frameworks

Domains of Expertise¶

  • Data Visualization: Creating clear, interactive charts and dashboards

  • Scientific Simulation: Implementing numerical methods for physics problems

  • Virtual Reality: Developing immersive experiences with real-world applications

  • Financial Analysis: Building tools for budget tracking and financial insights

Education & Research¶

My academic background includes work in:

  • Computational physics and materials science

  • Time-dependent simulations of superconducting systems

  • Virtual reality interface design and user experience research

  • Data analysis methodologies and visualization techniques

Philosophy¶

  • Open Source: Sharing knowledge and tools with the broader community

Current Interests¶

I’m currently exploring:

  • Advanced visualization techniques for scientific data

  • Machine learning applications in physics simulations

  • Improved user interfaces for VR applications

  • Automated financial analysis and reporting tools

Feel free to explore my projects to see these principles and technologies in action!
